How does “Activate on server” work?

After a successful payment, the customer receives one or more slots (subscriptions). In the Dashboard → Subscriptions tab, they choose a slot and link it to a Discord server where they have Admin/Manage Server permissions.

How does the bot verify premium?

Your bot calls the store API: GET /api/premium/:guildId?productId=... with header x-api-key. If it returns active: true, enable premium features.

Can I refund or cancel?

Use Stripe’s dashboard to refund payments. For cancellations, you can add logic later for recurring subscriptions. This template uses one-time payments for fixed durations.
